Service

데이터랩스는 블록체인 기반 서비스를 쉽고 빠르게 만들 수 있는 환경을 제공합니다.

블록체인 서비스를 보다 쉽게, BaaS

데이터랩스는 개발을 잘 알지 못해도 블록체인 서비스를 구현할 수 있도록 BaaS(Blockchain as a Service)를 제공합니다. BaaS는 클라우드 기반 솔루션을 통해 블록체인 서비스 운영에 필요한 대역폭 관리, 적당한 리소스 할당, 호스팅, 보안기능을 제공하여 블록체인을 쉽게 개발 할 수 있도록 간단한 툴을 제공해 드립니다.

블록체인 서비스를 보다 쉽게, BaaS

개발자를 위한 API 서비스

데이터체인 블록체인을 이용해 자신만의 서비스를 만들려면 네트워크의 노드를 운영해야 합니다. 블록체인 서비스를 시작한다는 것은 블록체인 네트워크의 데이터를 실시간으로 계속 동기화 해야 하기 고성능의 하드웨어를 갖추어야만 합니다. 데이터랩스는 블록체인 서비스에 필요한 개발 시간과 인력, 비용 절감을 API 서비스를 통해 해결하였습니다. 노드 구축, 기능을 직접 구현할 필요가 없이 보다 나은 블록체인 서비스에 집중할 수 있도록 하고, 계정 키를 관리할 필요가 없어 보안 이슈로부터 자유로운 개발을 할 수 있는 환경을 제공해 드립니다.

    개발 시간 단축

  • 노드를 직접 운영하지 않아 온전한 개발에 집중

    편의성

  • REST API, SDK로 데이터체인에서 지원하는 모든 타입의 트랜잭션 전송 가능
  • ERC-20, ERC721 토큰 거래내역 조회

    계정 키 관리

  • 데이터체인 계정의 개인 키 유출이 불가능한 구조를 사용
  • 독자적인 데이터체인 지갑을 쉽게 개발 가능

    비용 절약

  • 노드 설치가 필요없어 고성능의 하드웨어가 필요하지 않음
  • 블록체인 서비스 개발에 필요한 인력, 비용 절약
